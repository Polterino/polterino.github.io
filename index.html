PWA Update

From now on this website can be installed on desktop and mobile as a Progressive web app. In this way the app stores files on the device meaning it can be used when there's no internet connectivity. Remember that only supported browser and operating systems support PWAs, please refer to this image. If you see the "Install app" button you can use it, no more actions required. If you're on iOS (iPhone or iPad) and you use Safari, you must install it by clicking on the "Share" icon and then "Add to Home Screen".

Understanding Quiz Results

If a question has a green checkmark, it means that the answer the site gives as "correct" is 100% correct, because it has been verified. However, if there is a yellow warning sign, it indicates that the marked answer may not be correct, although it is likely to be so. Please pay attention to these symbols when reviewing your quiz results.
